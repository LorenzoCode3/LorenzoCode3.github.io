# Personal Website

This repository contains a personal website developed with Astro and styled with Tailwind CSS. It is distributed via GitHub Pages.

## Technologies

- Astro
- Tailwind CSS
- JavaScript / TypeScript

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the build:

```bash
npm run preview
```
