# Personal Website

This repository contains a personal website developed with Astro and styled with Tailwind CSS. It is distributed via GitHub Pages.

## Technologies

- Astro
- Tailwind CSS
- JavaScript / TypeScript

## Local Development

1. Install dependencies:

```bash
npm install
```

1. Start the development server:

```bash
npm run dev
```

1. Build for production:

```bash
npm run build
```

1. Preview the build:

```bash
npm run preview
```
